# Style Guidelines

## CSS
### Property Ordering:
```css
    .example {
        /* Positioning */
        position: absolute;
        z-index: 1;
        top: 0;
        right: 0;
        bottom: 0;
        left: 0;

        /* Display & Box Model */
        display: block;
        overflow: hidden;
        box-sizing: border-box;
        width: 600px;
        height: 250px;
        margin: 3px;
        border: 1px solid #222222;
        padding: 10px;

        /* Color */
        background: #222222;
        color: #FFFFFF;

        /* Text */
        font-family: sans-serif;
        font-size: 14px;
        line-height: 1.5;
        text-align: right;

        /* other */
        cursor: pointer;
        transition: margin-right 4s ease-in-out;
    }
```
